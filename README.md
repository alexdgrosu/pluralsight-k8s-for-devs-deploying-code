# Pluralsight // Kubernetes for Developers: Deploying Your Code

Personal repository for following along with the *Kubernetes for Developers: Deploying Your Code* course offered by *Pluralsight*. (https://www.pluralsight.com/courses/kubernetes-developers-deploying-code)

*How do you deploy code to Kubernetes? Learn about different deployment techniques including, rolling updates, Canary deployments, Blue-Green deployments, jobs and CronJobs, as well as troubleshooting techniques that can be used to identify issues.*
